Welcome to FZEE Flasher(beta)! The -UNOFFICAL- Flipper Zero ESP32 Dev board web flasher.

This site is to help facilitate flash updating ESP32 Dev Boards with custom firmware for the Flipper Zero. to use this web tool do the following;

  1. Put your ESP32 Dev Board in bootloader mode (some boards will not need this)
  2. Select the connect button at the top right of the page. (a box will show up asking you to connect the paired COM device hit connect)
  3. Next a Select Model menu will show up. using the drop downs select (board) (version) and (firmware).
  4. Next you will select (program). We have provided an (erase) function in case you feel there is a problem with the ESP32.
  5. Once the flash process is complete your ESP32 board will be ready to use with its new firmware.

FZEE Flasher FAQ:

Q: Is FZEE Flasher affiliated with Flipper Devices or any other custom hardware sites?
A: No, this site was built for the community by the community.

Q: Why can I not connect my ESP32 and get the error browser not supported?
A: Some Browsers have disabled serial support (for security) and will not allow a connection to the ESP32 board. You will need to enable it in the browser or use a browser that supports it.

Q: Will you support X firmware and when?
A: FZEE Flasher is very new and in beta. We do intend to support many poplar firmware's in the near future including ones from some 3rd party hardware venders.
